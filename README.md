# Flow Local

Приватная локальная копия Wispr Flow. Весь звук и текст остаются на твоём Mac — сеть нужна один раз, чтобы скачать модель.

**Как пользуешься:** держишь **Fn**, говоришь (RU/EN, можно вперемешку), отпускаешь — текст вставляется в курсор в любом приложении. В меню-баре — иконка статуса (🎙 готов / 🔴 запись / ✍️ распознаю).

## Дашборд (как у Wispr Flow)

Меню-бар → **«Открыть Flow Local…»** — нативное окно с разделами:

1. **Главная** — история всех диктовок: время, текст, куда вставлено, слов/мин; поиск, копирование, удаление. Обновляется живьём.
2. **Статистика** — всего слов, диктовок, средний темп, серии по дням/неделям, график за 14 дней, топ приложений.
3. **Словарь** — свои слова/имена/термины. Дают модели подсказку при распознавании, а «ослышки» (напр. «виспер флоу») автоматически заменяются на правильное написание.
4. **Сниппеты** — фраза-триггер → готовый текст. Скажи «моя подпись» — вставится вся подпись.
5. **Черновик** — заметки с автосохранением.
6. **Настройки** — клавиша, модель, язык, звуки, индикатор, история. Правки применяются сразу (клавиша/модель — после перезапуска, кнопка есть в окне).

Данные: `~/.flow-local/flow.db` (SQLite) и `scratchpad.txt`. Дашборд работает только на 127.0.0.1 с одноразовым токеном — снаружи недоступен.

## Установка (один раз)

```bash
bash install.sh
```

Скачает зависимости и модель whisper-large-v3-turbo (~1.5 GB). На Apple Silicon работает через MLX (GPU) — транскрибация быстрее реального времени.

### Разрешения (обязательно)

Настройки → Конфиденциальность и безопасность:

1. **Микрофон** → включить для Terminal
2. **Универсальный доступ** → включить для Terminal
3. **Мониторинг ввода** → включить для Terminal

Плюс: Настройки → Клавиатура → **«При нажатии клавиши Fn»** → **«Ничего не делать»** (иначе Fn будет открывать системную панель).

## Запуск

```bash
bash run.sh
```

Первая загрузка модели в память ~5-15 сек, иконка сменится с ⏳ на 🎙.

## Настройка

`~/.flow-local/config.json`:

```json
{
  "hotkey_keycode": 63,
  "model": "large-v3-turbo",
  "language": null,
  "min_duration_sec": 0.4,
  "sounds": true
}
```

Параметры: `hotkey_keycode` — 63=Fn, 61=правый Option, 54=правый Cmd; `model` — large-v3-turbo / large-v3 / medium / small; `language` — null=авто (RU/EN), "ru"=только русский. Комментарии в JSON не пиши — файл сломается.

## Автозапуск (опционально)

```bash
cp com.flowlocal.plist ~/Library/LaunchAgents/
# сначала поправь путь к папке внутри plist
launchctl load ~/Library/LaunchAgents/com.flowlocal.plist
```

## Приватность

1. Аудио пишется в память, на диск не сохраняется
2. Распознавание — целиком на устройстве (MLX/CTranslate2)
3. Никакой телеметрии, никаких запросов в сеть после установки
4. Буфер обмена восстанавливается после вставки

## Если что-то не так

1. **Иконка ⚠️ «Нет доступа»** — не выданы разрешения (пп. 2-3 выше), после выдачи перезапусти
2. **Fn не реагирует** — проверь «При нажатии Fn → Ничего не делать»; либо смени hotkey_keycode на 61 (правый Option)
3. **Текст не вставляется** — Универсальный доступ не выдан терминалу
4. **Медленно** — поставь `"model": "medium"` в конфиге
