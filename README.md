# Flow Local

Приватная локальная диктовка для macOS — альтернатива Wispr Flow без облака. Держишь клавишу, говоришь (RU/EN вперемешку), отпускаешь — текст вставляется в курсор в любом приложении. Весь звук и текст остаются на твоём Mac.

**Актуальная версия — v2 (Swift)** в папке [`v2-swift/`](v2-swift/): нативное приложение, whisper.cpp с Metal, ~50 МБ, память ~300–700 МБ. Старая Python-версия лежит в [`legacy-v1/`](legacy-v1/) как архив.

## Возможности

1. Push-to-talk (держишь клавишу) и hands-free (короткий тап = замок)
2. Распознавание целиком на устройстве: whisper large-v3-turbo (Metal на Apple Silicon)
3. Плавающий индикатор записи с живым waveform
4. Дашборд: история с поиском, статистика, словарь, сниппеты, черновик, настройки
5. Обучение словаря: правишь вставленный текст — приложение предлагает замену «ослышка → слово»; правки в истории и передиктовки тоже учитываются
6. Ноль телеметрии, сеть нужна один раз — скачать модель

## Требования

- macOS 13.3+ (Apple Silicon рекомендуется; на Intel распознавание заметно медленнее)
- ~600 МБ на модель

## Установка из готовой сборки (без компиляции)

1. Скачай `FlowLocal-*.zip` из [Releases](https://github.com/rookeed/wflow/releases), распакуй, перенеси `FlowLocal.app` в Программы.
2. Сборка не нотаризована, поэтому macOS её заблокирует. Сними карантин:
   ```bash
   xattr -rd com.apple.quarantine /Applications/FlowLocal.app
   ```
3. Скачай модель (~574 МБ):
   ```bash
   mkdir -p ~/.flow-local/models && curl -L -o ~/.flow-local/models/ggml-large-v3-turbo-q5_0.bin \
     https://huggingface.co/ggerganov/whisper.cpp/resolve/main/ggml-large-v3-turbo-q5_0.bin
   ```
4. Запусти FlowLocal, выдай права (см. ниже).

## Сборка из исходников

Нужен Xcode (или Command Line Tools со Swift 5.9+):

```bash
git clone https://github.com/rookeed/wflow.git && cd wflow/v2-swift
bash download_model.sh      # модель → ~/.flow-local/models
bash build_app.sh           # → build/FlowLocal.app (whisper.cpp подтянется сам, ~45 МБ)
cp -R build/FlowLocal.app /Applications/
```

Подпись: скрипт сам возьмёт твой сертификат Apple Development, если он есть (бесплатный Apple ID в Xcode → Accounts → Manage Certificates), иначе подпишет ad-hoc. С ad-hoc подписью каждая пересборка сбрасывает права — с сертификатом нет.

## Права (первый запуск)

Настройки → Конфиденциальность и безопасность — для **FlowLocal**:

1. **Микрофон** — спросит сам при первой записи
2. **Универсальный доступ** — для вставки текста (приложение само покажет диалог)
3. **Мониторинг ввода** — для горячей клавиши

После выдачи прав перезапусти приложение. Если хоткей Fn: Настройки → Клавиатура → «При нажатии клавиши Fn» → «Ничего не делать».

## Использование

- Иконка в меню-баре: ⌛ загрузка → 🎤 готов
- **Держи клавишу** (по умолчанию правый Option, меняется в настройках) — говори — отпусти: текст вставится в курсор
- **Короткий тап** — режим hands-free (🔒), ещё тап — стоп
- Меню-бар → «Открыть Flow Local…» — дашборд
- Поправил вставленный текст? Через несколько секунд появится предложение добавить исправление в словарь

## Данные и приватность

Всё в `~/.flow-local/`: `config.json`, `flow.db` (история/словарь/сниппеты), `models/`, `scratchpad.txt`, `log-v2.txt`. Аудио живёт только в памяти, на диск не пишется. Никаких запросов в сеть после установки.
