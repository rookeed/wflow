# Flow Local v2 (Swift, нативный)

Полный рерайт v1 (Python) на Swift. Тот же UX: держишь клавишу — говоришь — текст вставляется в курсор. Распознавание через **whisper.cpp + Metal**, целиком на устройстве.

## Что нового против v1

1. Один нативный .app без Python/venv: ~50 МБ на диске вместо ~2 ГБ окружения
2. Память: ~300–700 МБ (Q5-модель) вместо ~1.5 ГБ
3. Нормальная подпись и права: разрешения выдаются приложению Flow Local, а не Terminal
4. whisper.cpp с Metal — загрузка модели ~1 с, распознавание быстрее реального времени
5. Совместимость с v1: те же `~/.flow-local/config.json` и `flow.db` — история, словарь и сниппеты подхватываются (словарь-подсказки и автозамены работают)

Скоуп ядра (этап 1): хоткей (PTT + замок), запись, распознавание, вставка, оверлей-пилл с waveform, меню-бар, история/словарь/сниппеты из SQLite. Дашборд на SwiftUI — этап 2.

## Сборка (один раз)

```bash
bash download_model.sh      # ggml-large-v3-turbo-q5_0.bin, ~574 МБ
bash build_app.sh           # → build/FlowLocal.app
cp -R build/FlowLocal.app /Applications/
```

Первая сборка скачает whisper.cpp XCFramework (~45 МБ) через SPM.

## Разрешения (первый запуск)

Настройки → Конфиденциальность и безопасность — для **Flow Local**:

1. **Микрофон** — спросит сам при первой записи
2. **Универсальный доступ** (Accessibility) — для вставки Cmd+V
3. **Мониторинг ввода** (Input Monitoring) — для хоткея

Плюс, если хоткей Fn: Настройки → Клавиатура → «При нажатии клавиши Fn» → «Ничего не делать».

## Конфиг

`~/.flow-local/config.json` — тот же, что у v1. Отличия:

- `model`: `large-v3-turbo-q5` (по умолчанию) / `large-v3-turbo` / `large-v3` / `medium` / `medium-q5` / `small`. Файл модели ищется в `~/.flow-local/models/`, докачка — `bash download_model.sh <имя>`
- `hotkey_keycode`: 63=Fn, 61=правый Option, 54=правый Cmd (как в v1)

## Если что-то не так

1. **⚠️ «Нет доступа»** — выдай Универсальный доступ и Мониторинг ввода для Flow Local, перезапусти
2. **⚠️ «модель не найдена»** — `bash download_model.sh`
3. **Текст не вставляется** — Универсальный доступ не выдан
4. **После пересборки снова просит права** — ad-hoc подпись меняется; собери с `CODESIGN_ID="Apple Development: ..."` (см. `security find-identity -v -p codesigning`)
5. **`import whisper` не собирается** — проверь, что SPM скачал артефакт: `swift package resolve`, затем `ls .build/artifacts`
6. Лог: `~/.flow-local/log-v2.txt`

## Приватность
